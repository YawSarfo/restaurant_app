# Tasty Bat-El — Restaurant Website

A full-stack-style restaurant website built with HTML, CSS, JavaScript, Bootstrap, Tailwind CSS, and Webpack.

## Features

- Customer menu with food ordering
- Customer registration and login
- Shopping cart and checkout
- Admin panel with dashboard, order management, menu management, and contact messages inbox
- Contact form in footer
- Responsive design (mobile and desktop)

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript (ES6 modules)
- Bootstrap 5
- Tailwind CSS v4
- Webpack 5 (bundler)
- Babel (transpiler)
- localStorage for data persistence

## How to Run

1. Make sure Node.js is installed
2. Open a terminal in this folder
3. Run: npm install
4. Run: npm start
5. Open: http://localhost:3000

## Admin Access
